ScrollIt.js

Create easy and fast way their Single Page applications, no need JQuery or any other framework.

It's Cool! look it

Compatible for all moderns bowsers and opcional polyfill incluse for ie8 and ie7

Fast and compatible with mobile devices

Chrome / Firefox / Opera / Safari / IE 9+

Instalation on Git 
                         $> git clone https://github.com/andreymdias/scrollIt.git
Archives javascript and Stylesheets 
                        <link rel="stylesheet" href="scrollIt/scrollIt.min.css" ></link>
                        <!-- Optional Polyfill -->
                        <script src="javascripts/polyfill.js"></script>
                        <script src="scrollIt/scrollIt.js"></script>
Basic in HTML 
                        <body>
                        <section data-page='whateverSectionName1'></section>
                        <section data-page='whateverSectionName2'></section>
                        <section data-page='whateverSectionName3'></section>
                        <section data-page='whateverSectionName4'></section>
                        </body>

                        var ScrollIt = new ScrollIt();
